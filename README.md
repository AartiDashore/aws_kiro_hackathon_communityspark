## Getting Started

**Prerequisites:** Python 3.11+

```bash
# 1. Clone the repo
git clone https://github.com/your-username/communityspark.git
cd communityspark

# 2. Create and activate a virtual environment
python -m venv venv

# Mac/Linux
source venv/bin/activate
# Windows
venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Start the server
uvicorn main:app --reload
```

Open [http://localhost:8000](http://localhost:8000) вЂ” sample data seeds automatically on first run.

## Pages

| Route | Description |
|---|---|
| `/` | Home вЂ” browse and search businesses |
| `/deals` | Flash deals with live countdown timers |
| `/register` | Register a new business |
| `/business/{id}` | Business profile with reviews and deals |
| `/docs` | Interactive API docs (auto-generated by FastAPI) |

## API Overview

All endpoints are prefixed with `/api`.

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/api/businesses` | List all businesses |
| `POST` | `/api/businesses` | Register a new business |
| `GET` | `/api/businesses/{id}` | Get a business with its deals and reviews |
| `POST` | `/api/deals` | Post a flash deal |
| `GET` | `/api/deals` | List all active deals |
| `POST` | `/api/reviews` | Submit a review |

Full interactive docs available at `/docs` when the server is running.
